# Medios para el README

Carpeta para las capturas y el video/GIF de demostración referenciados en el
[`README.md`](../../README.md) raíz. Aún no se han añadido los archivos reales;
suéltalos aquí con estos nombres y el README los mostrará automáticamente.

| Archivo | Qué mostrar |
|---------|-------------|
| `timer.png`    | Pantalla principal de la faena (horizonte cielo/tierra, compañero capibara) |
| `break.png`    | Descanso con la fauna del Llano y su dato curioso |
| `settings.png` | Panel de ajustes |
| `stats.png`    | Estadísticas y rachas |

## Demo (GIF o video)

- `demo.gif` — clip de 10–15 s del flujo: iniciar faena → fin de faena (suena el
  aviso) → aparece la fauna en el descanso → volver a la faena.
- Para video MP4: súbelo a un Release o issue de GitHub y pega la URL en el
  README (GitHub lo reproduce embebido).

### Cómo capturar rápido

- **macOS:** `⇧⌘5` para grabar pantalla/área; exporta a GIF con cualquier
  conversor, o usa Kap (kap.app).
- **Windows:** herramienta **Recortes** (captura) y **Xbox Game Bar** (`Win+G`,
  graba video).
